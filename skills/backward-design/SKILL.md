# Backward Design

Use this skill when the teacher asks for a full lesson or unit outline.

## Workflow
1. Define specific learning outcomes.
2. Define observable success criteria and evidence.
3. Plan instructional sequence aligned to outcomes.
4. Add checks for understanding and differentiation notes.

## Output Format
- Outcomes
- Success criteria
- Lesson phases
- Assessment checks
- Differentiation
